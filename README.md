# video-processing-trainer

# Мобильное приложение-тренажёр по обработке видеоинформации

Дипломный проект. Android-приложение для практического обучения студентов дисциплине «Технологии обработки видеоинформации».

## Что делает приложение

Позволяет изучать и применять на практике:
- Просмотр видео в собственном плеере
- Изучение материала
- Прохождение тестов
- Отслеживание результатов

Результаты работы отображаются мгновенно — студент сразу видит эффект своих действий.

## Технологии

- **Язык:** Kotlin
- **Архитектура:** MVVM
- **UI:** XML, Material Design 3, ConstraintLayout
- **Обработка видео:** OpenCV 4.8 (Java-биндинги)
- **Сборка:** Gradle, AGP 8.1.0
- **Минимальная версия Android:** API 29 (Android 10+)
